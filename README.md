# 🚀 Impacthon – TrustLens Dashboard

Impacthon is a lightweight frontend security dashboard prototype called **TrustLens** that simulates real-time API monitoring, anomaly detection, and trust scoring.

It demonstrates how modern dashboards can visualize system health, attack patterns, and AI-based confidence metrics in a clean and interactive way.

---

## 📌 Features

- 📊 Interactive Dashboard UI
- 📡 Live API Activity Monitoring (Simulated)
- 🚨 Anomaly Detection Alerts
- 📜 Trust History Section
- 🛡️ Dynamic Trust Score Display
- 🤖 AI Confidence Indicator
- ⚠️ Simulated Attack Detection (e.g., Brute-force on `/login`)

---

## 🖥️ Preview

The dashboard displays:

- Trust Score (e.g., 92)
- AI Confidence (e.g., 94%)
- System Status (Stable / Risk)
- Real-time attack alerts
- Risk Level Indicators
